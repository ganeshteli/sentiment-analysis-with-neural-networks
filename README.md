# Sentiment Analysis with Neural Networks

This project focuses on **financial sentiment analysis** using various machine learning and deep learning techniques.

- **Data Handling**: `Pandas`, `NumPy`
- **Visualization**: `Matplotlib`, `Seaborn`
- **NLP**: `NLTK`, `TensorFlow`, `Keras`
- **Model Evaluation**: `Scikit-learn`

- **Data Preprocessing**
- **Vectorization Techniques**
- **Models**:
   - **Dense Neural Networks (DNN)**: Simple feedforward models.
   - **RNNs**: Includes `SimpleRNN` and `LSTM` for sequence modeling.
- **Training**:
   - **Callbacks**: `EarlyStopping`, `ReduceLROnPlateau` for optimization.
   - **Caching & Prefetching**
- **Evaluation**:
   - Metrics: Accuracy, precision, recall, F1-score
   - Visualization: Training loss and accuracy over epochs
- **Inference**:
   - Predicts sentiment for new financial sentences
